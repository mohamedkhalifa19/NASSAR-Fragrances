## ✨ Overview

**Nassar Ruddy Perfumes** is a modern e-commerce platform designed for perfume enthusiasts. The platform combines elegant design, fast performance, and powerful management tools to create a seamless experience for both customers and administrators.

Whether you're searching for your signature fragrance or managing an entire perfume catalog, Nassar Ruddy delivers a refined and intuitive experience.

## 🛠️ Local Development

### 1️⃣ Clone Repository

```bash
git clone <repository-url>
cd nassar-ruddy
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Environment Variables

Create a `.env` file:

```env
DATABASE_URL=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

NEXTAUTH_SECRET=
NEXTAUTH_URL=
```

### 4️⃣ Run Prisma Migrations

```bash
npx prisma migrate dev
```

### 5️⃣ Start Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
